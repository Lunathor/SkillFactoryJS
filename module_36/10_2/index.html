Тест проверки промокода

Этот HTML файл загружает ваш JavaScript скрипт для проверки промокода.

Что произойдет:

1. Появится окно prompt() для ввода промокода

2. После ввода появится alert() с результатом проверки

Примеры промокодов для тестирования:

  • скидка
  • Скидка
  • СКИДКА
  • СкиДкА
  • скидка123
  • промокод

Откройте консоль разработчика (F12) для просмотра возможных ошибок.
