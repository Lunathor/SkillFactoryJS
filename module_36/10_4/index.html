Тест возраста с правильными окончаниями

Этот HTML файл загружает ваш JavaScript скрипт для вычисления возраста с правильными окончаниями.

Что произойдет:

1. Появится окно prompt() для ввода имени

2. Появится окно prompt() для ввода года рождения

3. После ввода появится alert() с результатом и правильным окончанием

Текущий год:

Правила русского языка для слова "год":

  • "год" - для чисел, заканчивающихся на 1 (но не 11): 1, 21, 31, 41...
  • "года" - для чисел, заканчивающихся на 2, 3, 4 (но не 12, 13, 14): 2, 3, 4, 22, 23, 24...
  • "лет" - для всех остальных: 5, 6, 7, 8, 9, 0, 11, 12, 13, 14, 15...
